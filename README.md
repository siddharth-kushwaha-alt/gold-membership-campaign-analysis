# Gold Membership Campaign Analysis

## Project Overview

This project analyzes customer demographics, purchasing behavior, and campaign response data to identify factors influencing Gold Membership acceptance.

## Dataset

- 2,234 Customers
- Customer Demographics
- Income Information
- Product Spending
- Purchase Channels
- Campaign Response

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI

## Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. SQL Analysis
5. Machine Learning
6. Power BI Dashboard
7. Business Recommendations

## Key Insights

- Only about 15% of customers accepted the Gold Membership offer.
- High-income customers showed significantly higher acceptance rates.
- High-spending customers were more likely to accept membership offers.
- Wine and Meat products generated the highest spending.
- Web purchases were the most frequently used purchase channel.
- Recently active customers showed stronger campaign response.

## Business Recommendations

- Focus membership campaigns on high-income customers.
- Target high-spending customers for loyalty programs.
- Use digital channels to improve campaign reach.
- Re-engage inactive customers through retention campaigns.
